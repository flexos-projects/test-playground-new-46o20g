<flex_block type="instructions">
This folder contains two types of files:

CORE DOCS (000-008): Holistic overview documents, one per spec type.
These are AI-generated summaries of all specs in their category.
They are regenerated when specs change significantly.
Numbered 000-008, matching the spec type numbering.

GUIDES (100+): Cross-cutting documents that span multiple spec types.
Use for: discussion summaries, mental models, decision records,
architectural notes, process documentation.
Numbered sequentially from 100.

Do NOT put individual specs here. Specs go in their type folder
(vision/, features/, pages/, etc.).
</flex_block>
